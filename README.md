# Reservation System

Sistema web de gestión de reservas desarrollado con **Spring Boot**, **Angular** y **PostgreSQL**. La aplicación utiliza una arquitectura cliente-servidor mediante una API REST para administrar las reservas.

## Tecnologías

- Java
- Spring Boot
- Angular
- PostgreSQL
- Maven
- TypeScript

## Estructura

```
reservation/
├── reservation-backend/
└── reservation-frontend/
```

## Instalación

### Clonar el repositorio

```bash
git clone https://github.com/jorgezarate15/reservation.git
cd reservation
```

### Backend

```bash
cd reservation-backend
mvn spring-boot:run
```

### Frontend

```bash
cd reservation-frontend
npm install
ng serve
```

## Tecnologías utilizadas

- **Frontend:** Angular
- **Backend:** Spring Boot
- **Base de datos:** PostgreSQL
- **Comunicación:** REST API
